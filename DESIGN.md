# 출동 로그 (가칭) — 소방안전교육 게임 설계 문서

> 참고 원작: [네이처크래프트(과학의숲)](../네이처크래프트/네이처크래프트_backup_20260707.html) — 존 게이트형 퀴즈 진행, 자원가공, 랜덤이벤트, 업적/트로피 구조를 뼈대로 참고하되, 완전 독립 신작으로 개발.
> 콘텐츠 원본: `2026불조심길라잡이.pdf` (한국화재보험협회, 1659p)

## 1. 컨셉 한 줄 요약

신입 소방관이 실제 출동 콜을 처리하며 계급이 오르고, 그와 함께 낡은 간이초소가 중앙 종합상황실로 성장하는 RPG형 성장 게임. 맵 위를 캐릭터가 직접 이동하며 화염 몬스터를 진압하는 액션이 핵심 동사다. 마을 꾸미기(타운빌더) 프레임은 의도적으로 배제.

## 2. 핵심 루프

1. **허브 화면**: 캐릭터 아바타(계급장) + 소방서 전경이 함께 보임. 계급 / 소방서 등급 / 안전지수 표시
2. **출동 콜 선택**: 존별 무전 콜 리스트에서 미션 선택
3. **맵 액션 미션**: 탑다운 미니맵에서 이동 → 몬스터 진압 (3장 참고)
4. **보상 획득**: 대응점수·EXP·장비재료
5. **성장 투자**: 포인트로 캐릭터 장비(헬멧·방화복·장갑·소방차) 업그레이드 + 존 클리어 시 소방서 시설(훈련장·정비고·상황실) 확장 → 진급 컷씬
6. **돌발 이벤트**: 사이렌 알림 → 제한시간 OX퀴즈
7. **챌린지 모드**: 119 종합상황실 — 전 영역 랜덤 문제 스트릭/콤보
8. **엔딩**: 4존 클리어 + 챌린지 최고기록 달성 → 특급 소방관 임명식 + 소방서 최종 승격

## 3. 맵 액션 전투 시스템

### 3.1 이동
- 클릭(탭) 이동과 방향키/WASD 이동 모두 지원
- `position:absolute` + 좌표 갱신 방식(네이처크래프트의 건물 드래그와 동일 계열 기술) → `requestAnimationFrame` 루프로 이동 애니메이션 처리 (신규 도입 요소)

### 3.2 전투 흐름
몬스터 클릭 → 진압 도구 선택 UI 팝업 → 공격 실행 → 몬스터 HP 누적 감소 → 0 되면 처치

### 3.3 데미지 공식
```
피해량 = 기본공격력(캐릭터 레벨/계급 비례) × 도구계수
도구계수 = 정답 도구 선택 시 보너스 배율(×2~3) / 오답 도구는 저효율(×0.2 내외)
```
- 몬스터는 고정 HP를 가지고 있어 여러 번 타격이 누적되는 방식 (즉사 없음)
- 레벨(장기 성장)과 지식(정답 도구 선택)이 함께 데미지에 기여 — 초반엔 지식이 승부를 가르고, 후반엔 레벨도 받쳐줘야 하는 밸런스
- 오답 도구는 단순 무효과를 넘어 **역효과**가 있는 경우도 존재 (아래 표의 전기/기름화재)
- 처치 시간이 길어질수록 몬스터의 "화재 확산" 카운터가 진행 → 시간 압박 요소
- 캐릭터가 몬스터 공격 범위에 닿으면 "화상 게이지" 상승, 일정 이상이면 강제 후퇴

### 3.4 몬스터 - 정답 도구 매핑 (Part02 화재원인 8종 기반, 예시)

| 몬스터 | 정답 도구 | 오답 시 결과 |
|---|---|---|
| 전기화재괴물 | 분말소화기 / 차단기 내리기 | 물 사용 → 역효과(감전, 몬스터 체력 회복 or 플레이어 피해) |
| 기름화재괴물(주방) | 방화모포(덮기) | 물 사용 → 역효과(폭발) |
| 가스누출괴물 | 가스밸브 잠그기 | 소화기만 사용 시 저효율 |
| 담뱃불 임프 | 물 / 소화기 | 저효율이지만 위험 없음 |
| 산불괴물 | 물 / 흙 덮기 | 방화모포 저효율 |

> 존 3, 4 콘텐츠(장비/특수시설) 기반 몬스터·도구 매핑은 PDF 세부 문항 추출 단계에서 확정 필요.

## 4. 성장 축 (2개, 동시 진행)

- **캐릭터 계급**: 소방사시보 → 소방사 → 소방교 → 소방장 → 소방위 → 특급 소방관(엔딩)
- **소방서 등급**: 간이초소 → 119안전센터 → 지역소방서 → 광역본부 → 중앙종합상황실

## 6. 랜덤 이벤트 & 업적

- **돌발 화재경보**: 맵 위 무작위 타이밍에 사이렌 알림 → 제한시간 OX퀴즈 (네이처크래프트 "잡초 퀴즈"의 대응 요소)
- **명예의 전당**: 업적 → 훈장으로 재해석 (골드 소화기, 산불진압왕, 완벽대피왕 등). 히든 훈장 포함, 소방서 내 전시 공간에 진열

## 7. 연결 방식

완전 독립 신작. 네이처크래프트와 세이브/자원/UI 공유 없음. 필요 시 추후 이스터에그 수준의 크로스레퍼런스만 고려.
